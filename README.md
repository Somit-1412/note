# note

A simple command-line notes app written in C.

## Features

- Add notes
- List notes
- Delete a note by number
- Clear all notes with confirmation

Notes are stored locally in `~/.notes`.

---

## Install

Requirements:
- gcc
- make
- git

```bash
git clone https://github.com/Somit-1412/note.git
cd note
make
sudo make install
